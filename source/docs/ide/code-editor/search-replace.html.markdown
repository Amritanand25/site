##Local
Search and replace is supported on the currently selected file. 

The hotkeys are defined in Preferences and the defaults are

	;Find.
	; Type: hotkey 
	key_find = Cmd+F

	;Find next.
	; Type: hotkey 
	key_find_next = Cmd+G

	;Find previous.
	; Type: hotkey 
	key_find_prev = Shift+Cmd+G

	;Replace.
	; Type: hotkey 
	key_replace = Cmd+Alt+F

	;Replace all.
	; Type: hotkey 
	key_replace_all = Shift+Cmd+Alt+F
