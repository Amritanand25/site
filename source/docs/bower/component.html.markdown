You should be able to find a `component.json` file in the root folder for a Bower component. This file contains important details about a component.

	{
	    "name":  "MyProject",
	    "version": "0.0.1",
	    "dependencies": {
	        "backbone": "latest",
	        "requirejs": "2.1.1"
	    }
	}
