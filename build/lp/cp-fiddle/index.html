Features Overview

Boxes

Each project you create gets its own Box server. This is an Ubuntu instance preinstalled with NodeJS, Ruby, Python, Git, Mercurial and SVN.

However, you can easily add more languages, databases and componentswith Codio Box Parts, described below.

Boxes boot instantly and shut down instantly. In fact, you won't be aware of them until you pop into a Terminal window.

Features Overview

Box Parts

Box Parts a really cool feature that let you rapidly install additional components into your project Box. You can add languages (PHP, Java, Go, Haskell etc.), databases (Mongo, MySQL, PostgreSQL, Redis), web servers (Nginx, Apache) and other back-end components.

Click here to see the full list of Box Parts. If you want us to add a new Box Part, click here press the 'New Issue' button, ask and ye shall be given (as soon as we can).

Blazingly Fast

You will be amazed at how fast both the Codio IDE and Codio Boxes are. Because Codio runs in the Cloud, many Box operations (Git, npm install, bundle - anything that is a Cloud->Cloud operation) are much faster than your local machine.
