      Codio in Education

      Roadmap

    • Codio is a Web IDE that has everything you need in one place to build and run HTML5+CSS+JS applications of any size. It's missing a few things but take a look at our Roadmap page and you'll see what we're looking to add in the coming months.

      Right now, you can build and run any HTML5+CSS+JS application. It's a great place to start playing with web development, because there is no need to set up any stack. Once we've added deployment targets, Git/Hg integration, build tasks, code completion and more, we think you'll be ready to dump your desktop tools.

      • No sign up needed

        You can use our IDE without even having to sign up.

      • Collaborate in real time

        Collaboration is at the heart of what we are buildling. Google Docs style imminent but we're about to add Git support and chat.

      • Any size project

        Codio is equally at home with very large applications as with tiny demos.

      • If it's public, it's free

        Public projects are and always will be free.

      • A complete stack

        We provide everything you need to build and deploy your front-end applications. No need to worry about 3rd party providers unless you want to.

      • Learning with codio

        Codio is great for anyone learning web development. You can start coding without installing anything and we're going announce some major learning features soon.

    • Codio in Education

    • We have major plans for Codio's role in Education. Watch this space for news as we are going to be announcing some very exciting stuff in the coming weeks.

  • Shakin' it up

    We think that there is a lot that Codio can do for education and we want to shake a few things up.

    Our blog post A Post-Curriculum approach to teaching programming is a mini manifesto on the way programming is taught and how we plan to play a part in fixing it.

    Codio:Annotations

    We are about to release a set of features that is the first assault, which will allow you to produce rich, interactive tutorials and code samples from directly within an IDE, not a long web page of lengthy text and static code samples. Although you need Codio to run your annotated samples, you don't need it to build them. You can use any IDE you like.

    Take a look at our Say it with a sample blog post that talks about how this will work and by April 25th we aim to have it all available within the Codio IDE itself.

    Crowdteaching and Curation

    We are espousing crowd-teaching the second assault. We think that the community can be a potent weapon in edcuation. There are already thousands of people who are creating great tutorial content and we feel sure that there are thousands more who are ready and willing to lend a hand.

    We are about to initiate a GitHub project that acts as a central repository for tutorials and code samples that help people learn. Together with Codio:Annotations, this repository will be a rich resource for people to draw on.

    As the quantity of content increases, things will start to get disorganized and so we, together with community participants, will put a powerful curation framework (with a rich UI) in place that provides a structure for all projects. Any tutorial project can be tagged, categorized, commented on and voted on. This will help people select the best tutorials and make them even better through feedback, forking and enhancements.

  • These are our starter features in our journey to build the perfect web development IDE.

      • Any Web Application

        Any HTML5+CSS+JS front-end, web site, mobile web app or code library works in our IDE. And we're adding HAML, SCSS, SASS etc. before too long.

      • Beautiful Code Editor

        Like many others, we're eternally grateful to the ACE project for the best web-based code editor in existence.

      • Develop, Run, Host

        Codio is not just an IDE. It also lets you run and deploy your applications on the Codio platform.

      • 100% Cloud-Based & Fast

        Everything runs in your browser and is blazingly fast. You can access and fork projects without even needing an account.

      • Highly Collaborative

        Codio allows any number of users to collaborate on real-time projects. Full Git/Hg integration is right around the corner.

      • Fork any Project

        Fork any public or anonymous project without creating an account.

      • Teams & Organizations

        You can group Codio users into teams, who can then be assigned read/write/admin access. Soon, teams will be a part of the special Organization account.

      • Import from Git, Hg and Zip

        Import from Git/Mercurial repos and ZIP files and your project can be run instantly within the IDE. Just press the Run button.

      • Anonymous Mode

        You can access the Codio IDE without even signing up. By bookmarking the URL any project you create will be only editable by you. Others can look at your code and even fork it.

      • Private & Public Projects

        Watch Codio in action by loading up some projects and running them in the IDE. Just load the app and run!
