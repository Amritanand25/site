How it works

Codio combines the best web based IDE and code editing capabilities with a dedicated back-end server 'Box' for each of your projects. Better still, you can have unlimited projects and Boxes for free for all public projects. Unlimited private projects and Boxes require a subscription.

Your Boxes come with Node.JS and Ruby preinstalled, but you can easily add languages, components and utilities to any project. And if we don't have a component you want, just request it to be added to our library.

Boxes & Box Parts

Each project you create gets its own Box server. This is an Ubuntu instance that comes with NodeJS, Ruby, Python, Git, Mercurial and SVN preinstalled.

Box Parts a really cool feature that let you install additional components into your project Box. You can add languages (PHP, Go, Haskell etc.), databases (Mongo, MySQL, PostgreSQL, Redis), web servers (Nginx, Apache) and much more. And if our Parts catalog doesn't have what you need, just ask us to add it.

Blazingly Fast

You will be amazed at how fast both the Codio IDE and Codio Boxes are. Because Codio runs in the Cloud, many Box operations (Git, npm install, bundle - anything that is a Cloud->Cloud operation) are much faster than your local machine.

And if you're wondering about the speed of our IDE, just try it out. Not only is it packed with a mass of productivity features, you'll think you're running a desktop app.
