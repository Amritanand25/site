Productivity

An IDE should be all about productivity, and Codio has a ton of features that hugely increase your productivity. Everything you expect from the best desktop IDE's and Code Editors, is all in Codio and in the cloud. Syntax highlighting for hundreds of languages, code completion, search & replace, TextMate-like snippets, keyboard shortcuts, code folding, multiple cursors, color preview/picker, inline preview, and so much more.

Brilliance

We think that Bower is brilliant, and an amazingly useful tool for managing third party dependencies, so we baked it right into Codio and made it incredibly easy to use. We'll also be adding support for Grunt and Yeoman very soon.

Javascript!

We all love Javascript don't we? Well just in case you don't, you can use CoffeeScript in your Codio projects instead! And how about SASS or Less? Just code away and Codio will take care of the rest.

Deployment

Deploying your code should not be hard. With Codio, you can do so with a single click, then carry on coding. Define any number of deployment targets, and deploy to FTP/SFTP, Parse or NodeJitsu. Or deploy onto Codio's hosting platform for zero-setup production hosting of your static apps.

Collaboration

Codio makes it unbelievably easy and fast when when working in teams. Each keystroke you make is seen by your team mates as you type it. It makes for an extremely collaborative environment.

Git

You're not a real IDE until you have source control integrated. Control and version your code using Git without leaving Codio. Pull from any remote Git repository, then commit, branch, merge and push it all back! It's all in the cloud, which means no setup or installation is needed, and it all just works!

Annotations

Introducing an amazing new way to create rich, interactive documentation for tutorials, APIs, developer documentation or anywhere you want code documentation to be a cut above the rest - and it's all right within the IDE and a part of your code! No complicated formatting or standards to understand.
