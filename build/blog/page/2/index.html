From the Front-line - #26

Happy Friday! ...again

  • Intention.js is a very interesting, but slightly strange Javascript library that offers a way to dynamically restructure your HTML in a responsive manner using HTML attributes. And that's why I don't like it. Ends up dirtying your HTML.

  • Flipload.js provides a nice way to flip your HTML elements to show a loading indicator. Works with pretty much any element, which is nice.

  • I've seen these Parallax style effects before, but this library demo's what it can do very nicely indeed!

  • Ariya Hidayat writes an interesting article all about searching with Array.some().

  • This article on CSS sprites vs. Data URI's analyses which is faster and more efficient, particularly on mobile devices.

My goal is to try to publish every week day, so I would appreciate any help. If you find anything that you think would deserve a place in future issues, please let me know.

Quarter of a century!

  • Love Handlebars? Love Swag! A huge and extremely useful collection of Handlebars helpers.

  • Need to generate a PDF or two? well you don't need the server anymore. jsPDF will let you generate PDF files in client-side Javascript.

  • MobileHTML is a nifty site similar to CanIUse, with a focus on mobile browsers.

From the Front-line - #24

My last before a long weekend. see ya Tuesday!

  • FlowType is a nice library to add to your responsive tool belt, as it will automatically adjust your font size and line height based on element width.

  • More responsiveness for you with this article by Miles Johnson about cascading CSS inheritance with responsive design.

  • Tympanus does it again with this amazing tutorial on how to create a multi-level push menu that slides in from the side of the page. We've seen these before, but the extra touches here make it unique.

  • If you have a need to show your users some sort of count, then check out favcount.js, which is a JS library that enhance your site's favicon with a dynamically changing count.

  • Add depth to any element with this jQuery plugin, that will add a nice long drop shadow.

  • Premailer should be in every developer's arsenal. Give it the contents of your HTML email newsletter and it will show you any warnings that may mean your email doesn't look right in the most popular mail clients.

  • TriDiv is just an awesome example of how far browser based technology has come and what you can do with CSS.

My goal is to try to publish every week day, so I would appreciate any help. If you find anything that you think would deserve a place in future issues, please let me know.
