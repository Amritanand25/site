From the Front-line - #4

Goodie, goodie, goodie...

  • The ever popular Bower just hit 1.0.0 Alpha. In fact, it released four alphas in the last 24 hours ago! Here's a summary of what's new including details of the rewriting efforts.

  • Here's a fun and useful list of Github patterns.

  • Another great article from David Walsh, this time about creating keyboard shortcuts with the open source Mousetrap Javascript library.

  • Instagram are making further steps into the browser by introducing Instagram Web Embeds, which is simply an easy way to embed Instagram photos onto web pages. Kinda' funny that they used images in their post to demo this.

  • Media queries are awesome! Media Queries are also easy to forget. @Media.info helps a little by showing you the media query code for the device you are looking at their site with.

My goal is to publish every week day, so I would appreciate any help. If you find anything that you think would deserve a place in future issues, please let me know.
