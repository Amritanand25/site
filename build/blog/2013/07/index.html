From the Front-line - #16

  • An in-depth primer into front-end tooling using Grunt, for an open source workflow.

  • PSD.rb is a Ruby library that can parse Photoshop PSD files. The library itself isn't front-end related, but what it does is.

  • Huxley is a test-like system for catching UI regressions. It watches your browse, takes screenshots, and then tells you when they change. Great idea!

  • Jordan Lewis has written a post about a bunch of techniques for mobile and responsive cross-browser testing at Envato.

  • How did I not know about this one? Ghostlab is a Mac app that provides synchronized testing for web and mobile. Open your app on a couple of devices, then watch each one mimic your actions in the other. Pretty damn cool!

  • 6 simple steps to create a hipster logo.

  • Another great little blog post from Alex MacCaw about preventing tab-close data loss.

My goal is to publish every week day, so I would appreciate any help. If you find anything that you think would deserve a place in future issues, please let me know.

From the Front-line - #14

Bigger and better than evaarrrr...

  • The guys at MailGun just released a nifty little API to allow you to validate email quickly and easily. Not so sure it's actually that hard to do, and requires an API, but could be useful - maybe.

  • Great collection of really useful Chrome DevTools snippets.

  • A very detailed article on the mad world of scoping in Javascript and Coffeescript.

  • Need to quickly wireframe your mobile app? Check out Pentotype, which has a great looking pencil-drawn UI.

  • Move.js is a nice little Javascript library "making CSS3 backed animation extremely simple and elegant" from the genius that bought you Express, Jade and Mocha.

  • I really like the look of jspm.io, which is a browser based module loader that will load any module format locally or even directly from NPM or Github - nice!

  • Bootstrap 3 makes an early appearance with its first release candidate.

  • The guys at Stripe have created an online store where you can buy Stripe branded clothing. But it's the tiny UI touches on this site that are simply brilliant. Add something to the shopping basket, checkout but leave the form blank to see a great way to provide feedback on an incorrect form submission.

Lovin' it...

  • VerbalExpressions is a very interesting alternative to regular expression matching in Javascript, by providing a DSL style API.

  • Ajax Replay is a great way to test your Ajax API calls by replaying your actual Ajax calls. If you know the VCR Ruby Gem, this is a similar thing but for Javascript.

  • Now this is freakin' awesome! In fact, they are Awesome devices built in HTML and CSS. No images to see here.

  • Bone.io is a lightweight framework for building high performance Realtime Single Page HTML5 Apps. I think we may have heard that one before, but this one seems to centred around WebSockets instead of traditional Ajax.

  • Nice article attempting to simplify and explain Ember.js's routing and rendering.
