From the Front-line - #32

  • Hubspot have released another open source lib called Pace, but this time for creating good looking progress indicators for page load and ajax navigation.

  • jq is a lightweight and flexible command-line JSON processor. Slice, filter, map and transform JSON!

  • Now this is cool! BackgroundCheck is a JS lib for automatically switching the colour of an element, depending upon the brightness of images behind it.

  • More awesome effects... Rainyday simulates raindrops on a window.

  • Flexbox is coming! In fact it's already arrived, and is slower gaining wider browser support. Solved by Flexbox is a site that demo's what you can do with CSS Flexbox.

  • Thanks in no small part to CoffeeScript, Fat Arrow functions are about to make their way in Javascript.
