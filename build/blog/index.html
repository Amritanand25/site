From the Front-line - #31

  • Vinci is a great looking HTML5 image editor, that runs in the browser.

  • Looking for some 21st century dialogs? Check out Vex, which is a JS library built by the guys at Hubspot.

  • Fireshell is a boilerplate and workflow tool. Basically a competitor to Yeoman, but seems to be much more streamlined and focused, which I like.

  • Fancy running Linux in your browser with nothing but Javascript? Well why wouldn't you?

  • We use RequireJS - a lot! So the five helpful tips for using RequireJS are good read.

  • The 2nd edition of Eloquent Javascript has been announced. So if you're looking for a good Javascript book, go and pledge to this crowd-funded campaign.

  • Oh look! Another CSS framework! Ah but it's different... sort of. ChocolateChip-UI is only for mobile, and is designed to give a native look.

  • And it wouldn't be right without mentioning iOS7. This article by Thomas Fuchs, is the first part in a series about how to create a web app that looks like an iOS7 native app.

My goal is to try to publish every week day, so I would appreciate any help. If you find anything that you think would deserve a place in future issues, please let me know.

From the Front-line - #30

  • Wanna know more about native notifications in the browser? Check out this post by Adam Fortuna.

  • Now this post on full-text search using only CSS is a neat trick, but probably not one you wanna use for real.

  • Looking for a responsive navigation for your next site design? Check out Responsive Nav.

  • breakOn is a nifty little Javascript snippet to add to your Chrome dev tools, that will cause a break upon a given event.

  • grunt newer is a Grunt task that will only run a task against files that have been modified since the last run.

  • HTML5 Boilerplate gets a new release. 4.3 includes a bunch of fixes, and updates several of its dependencies.

  • LoDash also gets a new release, and a major one at that. Version 2.0 get custom builds, and a few new methods, including _.curry.

  • Is WebRTC ready yet? Check out this site which shows a nice table of the latest browser support.

And finally...

  • JSFuck is a new language based on Javascript, that uses only six characters to write and execute code. Best name ever!
